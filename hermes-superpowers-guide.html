Overview

전체 로드맵

각 Phase가 뭘 하는 단계인지, 완료하면 뭘 할 수 있는지 한눈에 봐요.

Phase 목표 완료 후 할 수 있는 것 기간
1 — 기본기 Claude Code 구조 이해 공식 문서 읽고 기본 CLI 사용 3~5일
2 — Superpowers 7단계 워크플로우 체험 구조화된 방식으로 기능 개발 가능 3~5일
3 — CLAUDE.md 전역 컨텍스트 파일 정비 매 세션 일관된 행동 보장 2~3일
4 — Hermes 연동 버티컬별 커스텀 스킬 제작 MAKE/CONTENT/WORK 자동 라우팅 1주
5 — 자동화 Hooks + 파이프라인 완성 Slack→Claude→Obsidian 전자동 1주~
Why

연동하면 뭐가 달라지나요?

Hermes에 Superpowers를 붙이면 생기는 실질적인 변화

✅ 장점

  • 트윗·블로그 등 콘텐츠 퀄리티가 매번 일정해짐
  • 빠뜨리는 항목(CTA, SEO 키워드 등)이 자동으로 체크됨
  • Hermes 버티컬이 개념에서 실제 동작 시스템이 됨
  • 장시간 자율 실행 가능 (2시간+ 드리프트 없이)
  • 코드 작업 시 테스트→구현 순서가 강제되어 버그 감소

❌ 단점

  • 짧은 작업도 브레인스토밍부터 시작해 느려질 수 있음
  • 단계마다 토큰 소비 → Pro 한도 더 빨리 찰 수 있음
  • 초기 SKILL.md 세팅이 손이 많이 감 (1~2일)
  • 트윗 1개처럼 간단한 작업엔 오버엔지니어링
💡 현실적인 팁: Superpowers를 모든 작업에 쓰지 말고, 긴 블로그 포스트·코드 개발에만 적용하고 짧은 트윗 하나는 그냥 쓰세요.

4Subagent Dev태스크별로 서브에이전트 투입. 2단계 리뷰(spec 준수 → 코드 품질)

6Code Review자동 코드 리뷰. Critical 이슈 발견 시 진행 차단
7Finish테스트 통과 확인 후 merge / PR open / 폐기 선택
