이런 사람에게 추천

  • GPT에서 이미지를 한 장씩 만들고 저장하는 게 귀찮다
  • 미리캔버스 요소를 대량 제작한다
  • 스티커, 이모티콘, GIF 소재를 반복 제작한다

  • 폴더 단위로 결과물을 관리하고 싶다

Step 08
이미지 생성 테스트
8
AI 이미지 생성 기능 확인
AI 이미지 생성 기능을 사용할 수 있는지 확인해줘. 가능하면 귀여운 고양이 캐릭터를 투명 배경 PNG로 생성해줘.

이미지가 생성되면 준비 완료

Step 09
여러 장 한 번에 생성 — 직업 캐릭터 세트
9
폴더 생성 + 이미지 10장 일괄 생성
jobs 폴더를 만들고 AI를 사용해 직업 캐릭터 PNG 10장을 생성해줘. [공통 스타일] - PNG - 투명 배경 - 귀여운 3D 클레이 스타일 - 동일한 색감 - 동일한 비율 - 스티커 스타일 - 텍스트 없음 [직업] 디자이너 교사 사무직 건설직 요리사 나머지 5개는 알아서 선정 파일명은 한글로 저장해줘.
Step 10
미리캔버스 요소 제작용
10
회사 업무용 아이콘 50개
elements 폴더를 만들고 회사 업무용 아이콘 50개를 생성해줘. [스타일] - PNG - 투명배경 - 귀여운 클레이 스타일 - 동일한 디자인 시스템 [주제] 회의 승인 검토 보류 긴급 점심 퇴근 캘린더 메일 나머지는 알아서 확장
Step 11
인스타 스티커 제작용
11
감정 스티커 20종
stickers 폴더를 만들고 동일 캐릭터를 사용해서 감정 스티커 20종 생성 [공통] - PNG - 투명배경 - 스티커 스타일 - 동일 캐릭터 유지 [감정] 좋아요 멘붕 분노 피곤 졸림 감동 퇴근 야근 커피충전 나머지는 알아서 추가
Step 12
Giphy 등록용 GIF 소재 제작
12
주제별 3종 × 5개 = 총 15장
giphy_set 폴더 생성 투명배경 PNG 생성 주제 카페인 충전 졸림 멘붕 좋아요 야호 각 주제별 기본 과장 극단 총 15장 생성
Tips
실전 팁

좋은 프롬프트 구조

  1. 폴더 생성
  2. 공통 스타일 지정
  3. 대주제 지정
  4. 소주제 지정
  5. 파일명 규칙 지정

추천 활용처

  • 미리캔버스 요소 제작
  • 인스타 만화 스티커
  • Giphy GIF 소재
  • 블로그 삽화
  • PPT 아이콘 세트
  • 앱용 캐릭터 리소스
  • 웹사이트 일러스트 세트
Ex
프롬프트 예시
animals 폴더 생성 [공통 스타일] 귀여운 클레이 스타일 투명 배경 PNG [대주제] 동물 [소주제] 고양이 강아지 토끼 햄스터 나머지는 알아서 추가

핵심

핵심은 "한 장 생성 → 저장 → 반복" 작업을
"폴더 단위 일괄 생성"으로 바꾸는 것이다.

반복 생산이 많은 디자이너나 콘텐츠 제작자일수록 효과가 크다.
