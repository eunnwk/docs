1. 동작 원리

이 스크립트는 Playwright라는 도구를 사용합니다. Playwright는 사람이 마우스·키보드로 하는 것처럼 브라우저를 직접 조작합니다. 쉽게 말하면 "로봇이 대신 블로그 작성 창을 열고, 글을 입력하고, 이미지를 첨부하고, 발행 버튼을 누른다"고 생각하시면 됩니다.

▶
스크립트 실행 & 포스팅 선택 터미널에서 python 명령어 실행 → 올릴 글 번호 입력
🔐
네이버 로그인 자동으로 브라우저가 열림 → 직접 네이버 로그인 → 완료되면 자동 진행
✍️
제목·본문 자동 입력 reviews_맛집_2024이후.md 파일에서 내용을 읽어 자동 입력
🖼️
이미지 자동 삽입 images/ 폴더에서 이름 규칙에 맞는 파일 자동 탐색 후 업로드
📍
네이버 지도 삽입 (국내만) 국내 장소는 지도 검색 후 자동 삽입 / 해외는 자동 건너뜀
🚀
발행 태그 입력 후 발행 완료

2. 사전 준비

항목필요한 것확인 방법
Python 3.9+ 파이썬 인터프리터 python --version 또는 python3 --version
pip 파이썬 패키지 설치 도구 (Python 설치 시 자동 포함) pip --version
Google Drive 파일 동기화 앱 (데스크탑 앱 설치 필요) drive.google.com/drive/downloads
네이버 계정 rreview 블로그 관리자 계정 —
💡 Python이 없다면? python.org/downloads에서 설치하세요. Windows는 설치 중 "Add Python to PATH" 체크박스를 반드시 체크해야 합니다.

3. 설치

처음 사용하는 기기에서 딱 한 번만 하면 됩니다.

  1. 터미널 열기 Command(⌘) + Space → "터미널" 검색 → Enter
  2. 스크립트 폴더로 이동

    아래 명령어를 복사해 터미널에 붙여넣기 하세요.

    cd ~/Library/CloudStorage/GoogleDrive-내계정@gmail.com/내\ 드라이브/naver-archive
    📁 폴더를 찾기 어렵다면? Finder에서 naver-archive 폴더를 찾아 터미널 창에 드래그하면 경로가 자동 입력됩니다.
  3. Playwright 설치 
    pip3 install playwright
  4. Chromium 브라우저 설치

    Playwright가 내부적으로 사용하는 브라우저를 설치합니다. 용량이 약 200MB이므로 인터넷 연결을 확인하세요.

    playwright install chromium
  5. 설치 확인 
    python3 -c "from playwright.sync_api import sync_playwright; print('설치 완료 ✅')"

    "설치 완료 ✅" 가 출력되면 정상입니다.

  1. PowerShell 열기 Windows 키 → "PowerShell" 검색 → Enter
    ⚠️ 주의 "Windows PowerShell"을 여세요. "명령 프롬프트(cmd)"보다 기능이 좋습니다.
  2. 스크립트 폴더로 이동 
    cd "G:\내 드라이브\naver-archive"
    📁 드라이브 문자가 G:가 아니라면? 파일 탐색기에서 "Google Drive"가 어느 드라이브로 마운트됐는지 확인하세요 (E:, F: 등일 수 있습니다).
  3. Playwright 설치 
    pip install playwright
  4. Chromium 브라우저 설치 
    playwright install chromium
  5. 설치 확인 
    python -c "from playwright.sync_api import sync_playwright; print('설치 완료')"
🛡️ "실행 정책" 오류가 난다면? PowerShell을 관리자 권한으로 실행 후 아래 명령어를 한 번만 실행하세요. 
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

5. 스크립트 실행 방법

  1. 터미널에서 폴더 이동 
    cd ~/Library/CloudStorage/GoogleDrive-내계정@gmail.com/내\ 드라이브/naver-archive
  2. 스크립트 실행 
    python3 naver_blog_post.py
  3. 올릴 글 선택

    아래처럼 목록이 표시됩니다. 번호를 입력하세요.

    === 포스팅 목록 ===
1. 오부이용 후기 | 금호동 프렌치 레스토랑 3회 방문
2. 리츠런던 티 | 런던 최고의 애프터눈 티
...
9. 쉐누 후기 | 서울 파인다이닝

올릴 번호 입력 (쉼표로 여러 개, 'all' 전체): 1
  1. PowerShell에서 폴더 이동 
    cd "G:\내 드라이브\naver-archive"
  2. 스크립트 실행 
    python naver_blog_post.py
  3. 올릴 글 선택

    번호를 입력하거나, 여러 개는 쉼표로 구분, 전체는 all 입력

올릴 번호 입력: all     ← 전체 올리기

6. 실행 후 단계별 워크플로우

스크립트를 실행하면 아래 순서로 자동 진행됩니다. 사람이 직접 해야 하는 단계는 🙋 표시입니다.

8. 다른 기기에서 사용하기

구글 드라이브 덕분에 스크립트와 이미지·원고 파일은 이미 어디서든 접근 가능합니다. 새 기기에서 딱 한 번 환경 설치만 하면 됩니다.

  1. Google Drive 데스크탑 앱 설치 & 로그인 drive.google.com/drive/downloads 에서 설치 후 같은 계정으로 로그인하면 파일이 자동 동기화됩니다.
  2. Python 설치 (없는 경우) python.org/downloads에서 설치. Windows는 반드시 "Add Python to PATH" 체크.
  3. Playwright + Chromium 설치 
    pip3 install playwright
playwright install chromium
    pip install playwright
playwright install chromium
  4. 이제부터는 동일하게 사용 폴더 이동 후 python naver_blog_post.py (또는 Mac은 python3) 실행.
🔄 파일은 자동 동기화 원고(md 파일)나 이미지를 어느 기기에서 수정해도 구글 드라이브가 자동으로 동기화해줍니다. 항상 최신 버전을 사용할 수 있습니다.

9. 자주 묻는 질문 & 문제 해결

❓ 브라우저가 자동으로 닫혔어요

로그인 대기 시간(3분)이 초과되면 스크립트가 종료됩니다. 다시 실행하고, 브라우저가 열리면 바로 로그인하세요.

❓ "이어서 작성하시겠습니까?" 팝업이 떠요

이전 세션에서 임시저장된 글이 있을 때 나타나는 팝업입니다. 스크립트가 자동으로 "새 글 작성"을 클릭하도록 처리되어 있으므로 별도로 조작할 필요가 없습니다.

❓ 이미지가 업로드되지 않아요

다음을 확인하세요:

  • 파일이 images/ 폴더 안에 있는지
  • 파일 이름이 규칙에 맞는지 (예: 01_오부이용_1.jpg)
  • 파일 확장자가 .jpg 또는 .png인지

❓ Mac에서 python 명령이 안 돼요

Mac에서는 python 대신 python3를 사용하세요. pip 대신 pip3도 마찬가지입니다.

❓ Windows에서 "pip는 내부 명령이 아닙니다" 오류가 나요

Python 설치 시 "Add Python to PATH"를 체크하지 않은 경우입니다. Python을 재설치하거나, py -m pip install playwright로 대신 실행해보세요.

❓ 네이버 지도가 삽입되지 않아요

해외 장소(리츠런던, 고디바도쿄, 마르케시1824, 산트유스타치오)는 지도 삽입을 자동으로 건너뜁니다. 국내 장소인데도 안 된다면 장소 이름이 네이버 지도에서 검색되는지 직접 확인해보세요.

❓ 스크립트 실행 중 오류가 났어요

터미널에 출력된 에러 메시지와 screenshots/ 폴더의 마지막 스크린샷을 확인하면 어느 단계에서 문제가 생겼는지 파악할 수 있습니다.

💡 스크린샷으로 디버깅하기 문제가 생겼을 때 naver_post_test.py를 먼저 실행해보세요. 단계별 스크린샷을 저장해줘서 어느 단계에서 멈췄는지 쉽게 파악할 수 있습니다.
