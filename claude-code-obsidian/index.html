STEP BY STEP

단계별 설치 가이드

순서대로 따라하세요. 이미 완료된 단계는 건너뛰어도 됩니다.

01

터미널 열기

맥에 기본 설치된 터미널 앱 실행

터미널은 글자로 컴퓨터에 명령을 내리는 앱입니다. 처음엔 낯설지만 이 가이드에서는 명령어를 그대로 복사해서 쓰면 됩니다.

가장 빠른 방법 — Spotlight 검색

키보드 단축키
⌘ Command + Space → terminal 입력 → Enter
💡 확인 방법: 검은색이나 흰색 바탕에 텍스트만 있는 창이 열리면 성공입니다. 이 창이 터미널입니다.
02

Node.js 설치 확인

Claude Code 실행에 필요한 환경 확인

Node.js는 Claude Code가 동작하기 위해 필요한 프로그램입니다. 이미 설치되어 있을 수도 있으니 먼저 확인합니다.

설치 여부 확인

터미널에 입력
node -v
✓ v18.0.0 이상 숫자가 나오면 → 이미 설치됨. Step 3으로 이동하세요.
✕ command not found가 나오면 → 아래 방법으로 설치하세요.

Node.js 설치 방법

  1. 브라우저에서 nodejs.org 접속
  2. LTS 버튼 클릭해서 다운로드
  3. 다운받은 파일 실행 → 계속 → 설치 완료
  4. 터미널 새로 열고 다시 node -v 입력해서 확인
03

Claude Code 설치

터미널에 명령어 한 줄 입력

아래 명령어를 터미널에 복사해서 붙여넣고 Enter를 누르세요. 설치에 1~2분 정도 걸립니다.

터미널에 입력
npm install -g @anthropic-ai/claude-code
⚠️ 오류가 나는 경우
앞에 sudo를 붙여서 입력하세요:
sudo npm install -g @anthropic-ai/claude-code
비밀번호 입력 요청 시 맥 로그인 비밀번호를 입력합니다. (화면에 표시 안 되는 게 정상)
04

Anthropic 계정 로그인

Claude.ai 계정과 연결

아래 명령어를 입력하면 브라우저 창이 자동으로 열립니다. Claude.ai에 로그인하면 터미널로 돌아와 연결이 완료됩니다.

터미널에 입력
claude login
💡 성공 화면: "Welcome back [이름]!" 메시지와 함께 계정 이메일이 표시됩니다.
05

Obsidian Vault 폴더 연결

Claude가 읽을 폴더 위치로 이동

Claude Code는 터미널에서 실행한 폴더 안의 파일들을 읽습니다. Obsidian Vault 폴더 안으로 이동해야 합니다.

방법 1. 드래그 앤 드롭 (가장 쉬움)

  1. 터미널에 cd 입력 (cd 뒤에 스페이스 하나)
  2. Finder에서 Obsidian Vault 폴더를 터미널 창으로 드래그
  3. 경로가 자동 입력됨 → Enter

방법 2. 직접 입력

터미널에 입력
cd ~/Documents/Obsidian\ Vault

Claude Code 실행

터미널에 입력
claude
💡 성공 화면: "Welcome back!" 메시지와 함께 Vault 폴더 경로가 표시되고 › 프롬프트가 나타납니다.

💡 권한 요청이 뜨면: y 누르고 Enter. 처음 한 번만 나옵니다.
06

첫 번째 테스트

파일 생성 & 읽기로 연결 확인

› 프롬프트 뒤에 아래 내용을 그대로 붙여넣고 Enter를 누르세요.

테스트 1. 파일 생성

Claude Code 프롬프트에 입력
오늘 날짜로 "테스트메모" 라는 마크다운 파일을 만들고, 내용에는 내가 자주 하는 일 3가지를 예시로 넣어줘

테스트 2. 파일 읽기

Claude Code 프롬프트에 입력
내 Vault에 있는 파일 목록을 보여줘
✅ 연결 성공 확인: Obsidian 앱을 열면 Claude가 만든 파일이 보입니다. 파일 목록과 내용을 읽어온다면 완벽하게 연결된 것입니다.
07

Notion 메모 이전 (선택)

노션에 쌓인 메모를 Obsidian으로 가져오기

노션에 메모가 쌓여있다면 Obsidian으로 가져와 Claude가 분석하게 할 수 있습니다.

1. 노션에서 내보내기

  1. 노션 왼쪽 하단 Settings 클릭
  2. Settings → Export all workspace content
  3. 형식: Markdown & CSV 선택
  4. Export 클릭 → zip 파일 다운로드

2. Obsidian으로 복사

  1. 다운받은 zip 파일 압축 풀기
  2. 압축 풀린 폴더 안의 .md 파일들을 Obsidian Vault 폴더에 붙여넣기

3. 파일명 자동 정리 (Claude에게 시키기)

⚠️ 노션 export 파일명에는 뒤에 ID 코드가 붙습니다
예: 회의록 a1b2c3d4e5f6.md
Claude Code 프롬프트에 입력
모든 마크다운 파일명 뒤에 붙어있는 32자리 영문숫자 코드를 제거해줘. 파일명만 바꾸고 내용은 건드리지 마.

📋 분석 & 요약
최근 1주일간 작성된 메모들을 읽고 핵심 주제를 정리해줘
내 메모 중에서 서로 연관된 아이디어를 찾아서 연결해줘
UX 관련 메모들만 찾아서 요약 노트를 새 파일로 만들어줘
특정 주제에 대해 내가 어떻게 생각이 변해왔는지 흐름을 분석해줘
✍️ 글쓰기 초안
내 문체와 논리 구조를 파악하고 새로운 블로그 포스팅 초안을 써줘
특정 주제에 대한 내 메모들을 바탕으로 발표 자료 구조를 잡아줘
내 최근 1주일간의 메모를 읽고 오늘 할 일을 정리해줘
🗂️ 지식 관리
비슷한 주제의 파일들을 폴더로 분류해서 정리해줘. 먼저 어떻게 분류할지 제안만 해줘.
중복되거나 비슷한 내용의 메모들을 찾아서 합칠 수 있는 것들을 알려줘
내 Vault 전체 구조를 분석하고 개선 방안을 제안해줘

NOTES

알아두면 좋은 것들

이미지 & PDF: Claude Code는 .md 텍스트 파일은 완벽하게 읽지만, 메모 안에 삽입된 이미지나 PDF 내부 텍스트는 인식하지 못할 수 있습니다.
파일명 관리: 폴더 구조가 복잡하거나 파일명이 모호하면 엉뚱한 파일을 참조할 수 있습니다. 파일명에 핵심 키워드를 포함해두세요.
데이터가 자산: 메모가 쌓일수록 Claude의 분석 정확도가 높아집니다. Obsidian에 꾸준히 기록하는 것 자체가 AI 비서를 훈련시키는 과정입니다.
