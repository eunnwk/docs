Slack → Obsidian
자동화 구축 가이드

회사에서 Slack에 메모를 남기면 → 집 Obsidian에 자동으로 쌓이는 파이프라인

Slack #kom-inbox
→
Zapier
→
Google Drive
→
Obsidian / Inbox
Step 01
Google Drive 폴더 생성
1
개인 Google 계정으로 drive.google.com 접속
2
내 드라이브 최상단에 새 폴더 생성
폴더명: kom-inbox
3
LG Gram 파일 탐색기에서 동기화 확인
내 드라이브/ └── kom-inbox/ ← 여기 보이면 OK
Step 02
Obsidian ↔ Google Drive 심링크 연결
⚠ Vault 전체를 Drive에 올리지 않고, Inbox 폴더만 Drive와 연결하는 방식. 개인 정보(재무, 소설 등)는 로컬 전용으로 유지됨.
1
시작 메뉴 → cmd 검색 → 관리자 권한으로 실행
2
심링크 명령어 실행
mklink /D "C:\Users\eunhy\obsidian-komx2\Inbox" "G:\내 드라이브\kom-inbox"

성공 메시지:

C:\Users\eunhy\obsidian-komx2\Inbox <<===>> G:\내 드라이브\kom-inbox에 대한 기호화된 링크를 만들었습니다.
항목 값
Vault 경로 C:\Users\eunhy\obsidian-komx2
Drive 문자 G:\
3
Obsidian에서 Inbox 폴더 확인
obsidian-komx2/ └── Inbox/ ← 폴더 보이면 연결 성공
ℹ Obsidian은 기본적으로 .md 파일만 표시. Google Drive에서 test.md 파일 생성 후 Obsidian Inbox에 나타나면 완전 연결 성공.
Step 03
Zapier 자동화 설정
ℹ Make.com은 Gmail 개인 계정의 Google Drive 연결 시 OAuth 오류 발생 → Zapier로 대체. Zapier 무료 플랜: 월 100회 실행.
1
Trigger 설정 — Slack
  • zapier.com → Create → Zap
  • Trigger → Slack 선택
  • Event: New Message Posted to Private Channel
  • 개인 Slack 계정 연결
  • Channel: #kom-inbox
  • Trigger for Bot Messages: No
  • Choose where to start: From now on
2
Action 설정 — Google Drive
  • Action → Google Drive 선택
  • Event: Create File From Text
  • 개인 Google 계정 연결
항목 값
Drive My Google Drive
Folder kom-inbox
File Name [Ts Time].md
Convert to Document? False

File Content 템플릿:

--- created: [Ts Time] source: slack tags: [inbox] --- [Text]
3
Publish & Turn On
  • Test step → 파일 생성 확인
  • 우상단 Publish 클릭 → Publish & Turn On
Result
완성된 파이프라인
회사에서 Slack #kom-inbox에 메시지 작성 ↓ (자동, 15분 이내) Google Drive / kom-inbox / [날짜].md 생성 ↓ (Drive for Desktop 즉시 동기화) LG Gram Obsidian / Inbox / [날짜].md 등장 ↓ 집에서 Obsidian 열면 이미 쌓여 있음 ↓ Claude와 함께 검증 / 분류 / 정리 → kom-tank
Note
Zapier 무료 플랜 제한
항목 무료 플랜
Zap 개수 5개
월 실행 횟수 100회 (하루 약 3~4개)
실행 주기 15분마다 체크
초과 시 해당 월 자동화 중단, 다음 달 초 리셋
