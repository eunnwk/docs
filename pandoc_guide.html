1 Pandoc이란?

Pandoc은 다양한 문서 형식을 서로 변환해주는 무료 오픈소스 도구입니다. "문서계의 스위스 아미 나이프"라고 불릴 만큼 폭넓은 형식을 지원해요.

설치만 하면 터미널에서 명령어 한 줄로 변환이 완료됩니다. 예를 들어 Obsidian에서 작성한 Markdown 파일을 Word 문서(.docx)나 PDF로 즉시 바꿀 수 있습니다.

💡
이런 분에게 추천해요
Obsidian / Markdown 노트를 Word로 제출해야 할 때  ·  Word 문서를 Obsidian으로 가져오고 싶을 때  ·  자동화 파이프라인에서 문서 포맷을 통일하고 싶을 때

2 지원 변환 형식

입력 형식→출력 형식 (예시)
Markdown (.md)→Word (.docx), PDF, HTML, EPUB
Word (.docx)→Markdown, HTML, PDF
HTML→Markdown, Word, PDF
LaTeX (.tex)→Word, HTML, PDF
Typst→PDF, HTML
EPUB→Markdown, Word, HTML

3 Windows 설치 방법 Windows

3가지 방법 중 하나를 선택하세요. 방법 A (Winget)를 가장 추천합니다.

방법 A  —  Winget (명령어 한 줄) ✓ 추천

Windows 10 버전 1709 이상, Windows 11에서 바로 사용 가능합니다.

  1. 키보드에서 Windows 키 + R을 누른 뒤 powershell을 입력하고 Enter를 누릅니다.

  2. 아래 명령어를 복사해서 붙여넣고 Enter를 누릅니다.

    PowerShell
    winget install --id JohnMacFarlane.Pandoc

  3. 설치가 완료되면 PowerShell 창을 닫고 새로 열어주세요.

방법 B  —  설치 파일 직접 다운로드

명령어가 불편하거나 Winget이 없는 경우에 사용하세요.

  1. 브라우저에서 https://github.com/jgm/pandoc/releases/tag/3.10 으로 이동합니다.

  2. 페이지 아래 Assets 섹션에서 pandoc-3.10-windows-x86_64.msi를 클릭해 다운로드합니다.

  3. 다운로드된 .msi 파일을 더블클릭하여 설치합니다. '다음(Next)' 버튼만 계속 누르면 됩니다.

방법 C  —  Chocolatey

Chocolatey 패키지 매니저가 이미 설치되어 있는 경우에만 사용하세요.

PowerShell
choco install pandoc

4 macOS 설치 방법 macOS

2가지 방법 중 하나를 선택하세요. 방법 A (Homebrew)를 가장 추천합니다.

방법 A  —  Homebrew (가장 쉬움) ✓ 추천
📌
Homebrew가 없는 경우 — 먼저 설치하기
터미널에서 아래 명령어를 붙여넣고 Enter하세요. 설치 완료 후 아래 Pandoc 설치를 진행합니다.
Terminal
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Pandoc 설치

  1. 터미널(Terminal) 앱을 실행합니다. (Spotlight에서 'terminal' 검색)

  2. 아래 명령어를 입력합니다.

    Terminal
    brew install pandoc

이미 설치되어 있다면 업그레이드 명령어를 사용합니다.

Terminal
brew upgrade pandoc
방법 B  —  설치 파일 직접 다운로드
❓
내 Mac이 Apple Silicon인지 Intel인지 확인하는 방법
화면 왼쪽 상단 🍎 클릭 → '이 Mac에 관하여' 선택 → '칩' 항목에 Apple M1/M2/M3/M4 = Apple Silicon, Intel Core = Intel

  1. 릴리즈 페이지로 이동합니다.

  2. Assets 섹션에서 본인 Mac에 맞는 파일을 다운로드합니다.
    pandoc-3.10-arm64-macOS.pkg — Apple Silicon (M1/M2/M3/M4)
    pandoc-3.10-x86_64-macOS.pkg — Intel Mac

  3. 다운로드된 .pkg 파일을 더블클릭하여 설치합니다.

6 기본 사용법

Pandoc은 터미널 / PowerShell에서 아래 형식으로 사용합니다.

기본 형식
pandoc  입력파일  -o  출력파일

-o는 출력(output)의 약자입니다. 파일 확장자를 보고 형식을 자동으로 판단합니다.

7 변환 예시

① Markdown → Word

Obsidian 노트나 .md 파일을 Word(.docx)로 변환할 때 사용합니다.

Terminal / PowerShell
# 현재 폴더의 파일 변환
pandoc 내문서.md -o 내문서.docx

# 경로 지정 (Windows)
pandoc C:\Users\이름\Documents\노트.md -o C:\Users\이름\Desktop\결과.docx

# 경로 지정 (macOS)
pandoc ~/Documents/노트.md -o ~/Desktop/결과.docx
② Word → Markdown

Word 파일을 Obsidian이나 Markdown 편집기로 가져올 때 사용합니다.

Terminal / PowerShell
pandoc 보고서.docx -o 보고서.md
③ Markdown → HTML

웹 페이지용 HTML 파일을 생성합니다. -s 옵션으로 완전한 독립 파일을 만들 수 있습니다.

Terminal / PowerShell
pandoc 문서.md -s -o 문서.html
④ Markdown → PDF

PDF 변환은 별도 도구가 필요합니다. 간단한 방법은 HTML로 변환 후 브라우저에서 인쇄 → PDF 저장을 이용하세요.

Terminal / PowerShell
# HTML로 변환 후 브라우저에서 PDF 저장
pandoc 문서.md -s -o 문서.html

# wkhtmltopdf 설치 후 직접 PDF 변환
pandoc 문서.md -o 문서.pdf --pdf-engine=wkhtmltopdf
⑤ 여러 파일 합치기

여러 Markdown 파일을 하나의 Word 문서로 합칩니다.

Terminal / PowerShell
pandoc 1장.md 2장.md 3장.md -o 전체문서.docx

8 자동화 파이프라인 활용

Pandoc은 Python 스크립트나 자동화 도구와 함께 사용하면 더욱 강력해집니다.

Python 스크립트에서 호출하기

report-bot 같은 자동화 스크립트에 문서 변환 단계를 추가할 수 있습니다.

Python
import subprocess

# Markdown 파일을 Word로 자동 변환
subprocess.run([
    'pandoc',
    'report.md',
    '-o', 'report.docx'
])

Obsidian 노트 → 배포용 문서

Obsidian에서 작성한 노트나 보고서를 팀에 공유할 때 회사 템플릿 스타일을 그대로 적용할 수 있습니다.

Terminal / PowerShell
pandoc ~/obsidian-vault/EUT-보고서.md \
  -o ~/Desktop/EUT-보고서-배포용.docx \
  --reference-doc=template.docx

# --reference-doc 옵션으로 회사 템플릿(폰트, 색상 등) 적용 가능

9 유용한 옵션 모음

옵션설명
--toc목차(Table of Contents) 자동 생성
-s / --standalone완전한 독립 HTML 파일 생성 (CSS 포함)
--reference-doc=파일.docxWord 문서 스타일 템플릿 적용
-f / -t입력(-f) 및 출력(-t) 형식 명시
--wrap=none자동 줄바꿈 없이 긴 줄 유지
--pdf-engine=wkhtmltopdfPDF 변환 엔진 지정

10 FAQ / 문제 해결

'pandoc' 명령어를 찾을 수 없다고 나와요 ▾
설치 후 터미널 / PowerShell을 새로 열었는지 확인하세요. 이미 열려있던 창에서는 인식이 안 될 수 있습니다. 그래도 안 된다면 PC를 재시작해보세요.
PDF 변환이 안 돼요 ▾
PDF 변환은 별도 도구(LaTeX 또는 wkhtmltopdf)가 필요합니다. 가장 간단한 방법은 HTML로 변환한 후 브라우저의 '인쇄 → PDF 저장' 기능을 사용하는 것입니다.
한글이 깨져요 ▾
HTML 변환 시 인코딩 옵션을 추가해보세요. 
pandoc 문서.md -o 문서.html --metadata charset=utf-8
이미지가 안 보여요 ▾
이미지 경로가 상대 경로인 경우, 파일이 있는 폴더로 이동한 뒤 명령어를 실행하세요. 
# Windows
cd C:\Users\이름\Documents\내프로젝트
pandoc 문서.md -o 문서.docx

# macOS
cd ~/Documents/내프로젝트
pandoc 문서.md -o 문서.docx
