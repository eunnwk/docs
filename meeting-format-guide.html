Google Docs 회의록
AI 최적화 포맷 가이드

📅 2026-05-12 ⏱ 약 15분 🏷 Google Docs · Apps Script · 회의록

Google Docs에 회의록을 기록할 때 AI가 파싱하기 좋은 포맷으로 작성하면 자동 정제 품질이 크게 높아진다.

이 가이드는 최적 포맷을 정의하고, 구글 문서에서 해당 포맷을 손쉽게 입력할 수 있는 세 가지 방법을 단계별로 안내한다.

Format

AI 최적화 회의록 포맷

아래 포맷을 기준으로 회의록을 작성한다. 날짜 형식과 섹션 제목을 매번 동일하게 유지하는 것이 가장 중요하다.

21 Apr 2026 | 회의 제목
 
Attendees: 홍길동, 김철수, 이영희
 
[안건]
- 안건 1
- 안건 2
 
[논의]

- 압축하지 않아도 됨, 길게 써도 OK
 
[결정]
- 결정된 사항만 명확하게
- 없으면 "없음"
 
[액션아이템]
- 홍길동 : 디자인 시안 수정 (기한: 4/28)
- 없으면 "없음"
 
[이슈]
- 반복되거나 우려되는 사항
- 없으면 이 섹션 생략

Methods

포맷 적용 방법 3가지

상황에 맞게 선택한다. 방법 3이 가장 편하고 오류가 적다.

방법 1

템플릿 탭

탭 하나를 템플릿으로 만들어두고 복사해서 사용

방법 2

자동완성 단축어

단축어 입력 시 포맷 자동 삽입

방법 3 ★ 추천

Apps Script 버튼

버튼 클릭 한 번으로 오늘 날짜 포함 포맷 자동 삽입

Method 01

방법 1 — 템플릿 탭 만들기

가장 간단한 방법. 별도 설정 없이 지금 바로 시작할 수 있다.

장점
  • 설정 필요 없음
  • 지금 당장 가능
단점
  • 매번 복사해야 함
  • 날짜 직접 입력 필요
  • 회의록 구글 문서 열기
  • 하단 탭에서 + (탭 추가) 클릭
  • 탭 이름을 📋 템플릿 으로 변경 (탭 이름 더블클릭)
  • 아래 포맷을 그대로 붙여넣기

[액션아이템]

사용 방법: 새 회의록 작성 시 템플릿 탭에서 전체 선택(Cmd+A) → 복사(Cmd+C) → 해당 탭에 붙여넣기 → 날짜와 내용 입력

💡

날짜 형식 예시: 12 May 2026 — 월은 반드시 영문 3글자(Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec)

Method 02

방법 2 — 자동완성 단축어 등록

구글 문서의 환경설정에서 단축어를 등록해두면, 특정 단어를 입력할 때 자동으로 텍스트가 치환된다.

장점
  • 탭 안에서 바로 입력 가능
  • 설정 한 번으로 영구 사용
단점
  • 줄바꿈 포함 긴 텍스트 등록 불가
  • 날짜 직접 입력 필요
  • 섹션 제목만 단축어로 가능
⚠️

구글 문서 자동완성은 줄바꿈이 포함된 긴 텍스트 등록이 되지 않는다. 섹션 제목처럼 짧은 텍스트만 단축어로 등록할 수 있다.

단축어 등록 방법

  • 구글 문서 상단 메뉴 도구 클릭
  • 환경설정 클릭
  • 대체 탭 클릭
  • 아래 표를 참고해 단축어 등록
입력 변환
//안건[안건]
//논의[논의]
//결정[결정]
//액션[액션아이템]
//이슈[이슈]
✅

이후 구글 문서에서 //안건 입력 후 스페이스 또는 엔터를 누르면 [안건] 으로 자동 변환된다.

Method 03 ★

방법 3 — Apps Script 버튼 만들기

구글 문서 안에 버튼을 만들어서, 클릭하면 오늘 날짜가 자동으로 포함된 포맷이 삽입된다. 날짜 오타가 없고 가장 편하다.

장점
  • 버튼 클릭 한 번으로 완료
  • 날짜 자동 입력 (오타 없음)
  • 초기 설정 후 영구 사용
단점
  • 초기 설정 15분 필요
  • 처음 실행 시 권한 허용 필요

1. Apps Script 열기

  • 구글 문서 상단 메뉴 확장 프로그램 클릭
  • Apps Script 클릭 → 새 탭에서 편집기가 열림
  • 기본으로 작성된 코드 전체 삭제

2. 스크립트 붙여넣기

아래 코드를 편집기에 붙여넣기:

Apps Script (JavaScript) 
function 새회의록추가() {
  const doc = DocumentApp.getActiveDocument();
  const body = doc.getBody();

  // 오늘 날짜를 "21 Apr 2026" 형식으로 생성
  const months = ['Jan','Feb','Mar','Apr','May','Jun',
                  'Jul','Aug','Sep','Oct','Nov','Dec'];
  const today = new Date();
  const dateStr = today.getDate() + ' ' +
                  months[today.getMonth()] + ' ' +
                  today.getFullYear();

  // 포맷 텍스트
  const template = dateStr + ' | 회의 제목\n\n' +
    'Attendees: \n\n' +
    '[안건]\n- \n\n' +
    '[논의]\n- \n\n' +
    '[결정]\n- \n\n' +
    '[액션아이템]\n- 담당자 :  (기한: )\n\n' +
    '[이슈]\n- \n\n' +
    '---\n\n';

  // 문서 맨 앞에 삽입
  body.insertParagraph(0, template);

  DocumentApp.getUi().alert('✅ 새 회의록 포맷이 추가되었습니다!');
}

function onOpen() {
  DocumentApp.getUi()
    .createMenu('📝 회의록')
    .addItem('새 회의록 추가', '새회의록추가')
    .addToUi();
}

3. 저장 및 권한 허용

  • 상단 💾 저장 버튼 클릭 (또는 Cmd+S)
  • 프로젝트 이름 입력창이 뜨면 회의록 자동화 입력 → 확인
  • ▶ 실행 버튼 클릭 → 실행할 함수: onOpen 선택 → 실행
  • 권한 요청 팝업이 뜨면 권한 검토 → 본인 계정 선택 → 허용
⚠️

"Google이 확인하지 않은 앱" 경고가 뜨면: 고급 클릭 → "회의록 자동화"(으)로 이동 클릭 → 허용

4. 사용 방법

  • 구글 문서로 돌아가기 (탭 전환)
  • 문서를 새로고침 (Cmd+R) — 메뉴에 📝 회의록 탭이 생김
  • 상단 메뉴 📝 회의록 → 새 회의록 추가 클릭
  • 문서 맨 위에 오늘 날짜가 포함된 포맷이 자동 삽입됨
  • 제목, 참석자, 내용만 채우면 완료
🎯

이후부터는 구글 문서를 열고 📝 회의록 → 새 회의록 추가 만 클릭하면 된다. 날짜는 항상 자동으로 올바른 형식으로 입력된다.
